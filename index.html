Tentang saya

Nama saya Robby Takdirillah saya lahir di Sukabumi tiga Desember 1997 saat ini saya masih berstatus mahasiswa aktif jurusan teknik informatika semester tujuh saya tertarik mencari pengalaman baru di dunia pekerjaan selain untuk menambah pengalaman saya saat sudah lulus kuliah nanti dan juga untuk menambah relasi serta belajar bagaimana tantangan dan hambatan kehidupan di dunia kerja. Saya adalah seorang blogger selain dari tujuan monetasi ngeblog juga dijadikan ajang untuk berlatih pemrograman dan juga mencari pengetahuan baru dikarenakan dengan ngeblog diharuskan untuk uptodate. Saya juga sering membangun sebuah project web baik itu untuk sebuah perusahaan, sekolah ataupun untuk individu.

Pengalaman berorganisasi saya pernah menjadi bagian kabinet himpunan mahasiswa teknik informatika di divisi keagamaan, saya juga pernah masuk sebuah komunitas gerakan literasi di kampus yang bernama library lovers community disana saya menjabat sebagai divisi sumber daya manusia dan juga pernah terlibat sebagai ketua pelaksana di salah satu kegiantanya dan ada beberapa organisasi lain yang pernah saya ikuti.
